What is Synote Media Fragment Player

Synote Media Fragment Player is a cross-browser and cross-platform player to visualise Media Fragment URIs. The player is a client-side implementation of W3C Media Fragment URI 1.0. The initial work started from the Synote project, which is synchronised annotation system for audio and video replaying.

Given a media resource URI on the Web, Synote Media Fragment Player will automatically choose the suitable player to play back the audio or video resource based on the audio/video format, the browser and the platform. The fragment information will be detected and the visual output will be delivered on the Web page. We also define the commonly used APIs for developers to control the player, such as play, pause, stop, forward, rewind, getPosition, setPosition, etc. Currently, we only implement the temporal dimension. The spatial, track and name axes, which are defined in W3C Media Fragment URI 1.0 will be implemented later.

  • Supported file format: mp3, aac, m4a, ogg, wav, mp4, mov, f4v, flv, 3gp,3g2, ogv, webm, youtube.com, youtu.be, wma, wmv, avi, mpg, mpeg, m1v, mp2, mpa
  • Supported browser: IE 8+ (partially, some formats do not replay correctly), Firefox 4+, Latest Google Chrome, Latest Safari
  • Supported platform: Windows, Mac OS X, Linux
Important

Some players need special plugin or add-on for the browser. For example, to play wmv in Firefox, Sliverlight plugin will be needed.
