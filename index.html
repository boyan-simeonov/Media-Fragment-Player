Use Media Fragment with Ease!

  • A client-side implementation of W3C Media Fragment URI 1.0 (basic) Specification
  • Based on MediaElement.js, cross-platform and browsers
  • Play the temporal dimension of media fragments for online media YouTube, DailyMotion, Vimeo

Dependencies

  • MediaElement.js
  • Media fragment parser (by Thomas Steiner)
  • jQuery 1.7.1
  • jQuery url Plugin
  • jQuery timer Plugin
  • Base.js, version 1.1a
