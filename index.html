Use W3C Media Fragment with Ease!

  • A client-side implementation of W3C Media Fragment URI 1.0 (basic) Specification
  • Play both temporal and spatial fragment
  • Based on MediaElement.js, cross-platform and browsers
  • Bring media fragments to online video sharing platforms, like YouTube and DailyMotion
  • Test on various browsers and devices. See Test results
  • Support file formats: wma, m4a, mp3, wav, mpeg, mp4, m4v, mov, wmv, flv, ogg, webm
Download v1.0-alpha

Dependencies

  • MediaElement.js
  • Media fragment parser
  • jQuery 1.7.1+
  • jQuery url Plugin
